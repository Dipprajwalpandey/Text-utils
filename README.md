# TextUtils

TextUtils is a simple and efficient text analyzer web application built with Django. It allows users to perform quick text operations like word/character counting, case conversion, and removing extra spaces - all through a clean, easy-to-use interface.

## Features

- Convert text to uppercase or lowercase
- Remove extra spaces from text
- Word and character count
- Clean, responsive UI with a light, professional theme
- About and Contact pages

## Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS (custom light theme, Inter font)
- Database: SQLite (default Django dev DB)

## Getting Started

### Prerequisites
- Python 3.x
- pip

### Installation

1. Clone the repository
   git clone https://github.com/Dipprajwalpandey/Text-utils.git
   cd Text-utils

2. Install dependencies
   pip install -r requirements.txt

3. Run migrations
   python manage.py migrate

4. Start the development server
   python manage.py runserver

5. Visit http://127.0.0.1:8000/ in your browser
